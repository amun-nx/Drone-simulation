## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/amun_nx/Drone-simulation.git
cd Drone-simulation
```
### 2. Run the project
```matlab
run('Linear.m')
open('Drone_project.slx')
```
